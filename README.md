
#  King Sukh Guest House Website Redesign

A modern, responsive, and user-friendly redesign of the King Sukh Guest House website built using React.js and Tailwind CSS.


##  Live Preview

kingsukh-intern.netlify.app


## Project Overview

This project focuses on redesigning the existing guest house website to improve its visual appeal, usability, and booking experience.

The original website lacked modern UI, responsiveness, and a proper booking system. This redesign solves those issues and provides a clean, professional interface.


##  Objectives

* Improve UI/UX design
* Make website fully responsive
* Create a structured booking system
* Enhance user engagement with animations
* Strengthen brand presentation


##  Tech Stack

*  React.js
*  Tailwind CSS
*  React Router DOM
*  React Icons
*  AOS (Animate On Scroll)


##  Features

*  Full-screen image slider (Hero section)
*  Booking system with date selection
*  Booking page with form validation
*  About section with animations
*  Services section with icons
*  Rooms section with slider
*  Gallery showcase
*  Customer reviews with like feature
*  Fully responsive design

---

##  Problems Identified & Solutions

###  1. Weak Background Design

**Solution:** Added full-screen image slider with gradient overlay

###  2. Poor Logo Visibility

**Solution:** Redesigned navbar and improved branding

###  3. Unprofessional Booking (WhatsApp redirect)

**Solution:** Created dedicated booking page with form

###  4. Low User Engagement

**Solution:** Added animations, sliders, and interactive UI

###  5. Not Mobile Friendly

**Solution:** Implemented responsive design using Tailwind
